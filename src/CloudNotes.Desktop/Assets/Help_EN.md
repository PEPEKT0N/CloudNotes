## Markdown Syntax

### Text Formatting

| Syntax | Result |
|--------|--------|
| `**bold**` | **bold** |
| `*italic*` | *italic* |
| `~~strikethrough~~` | ~~strikethrough~~ |

### Headings

Write `#` for heading 1, `##` for heading 2, `###` for heading 3.

### Lists

Write `-` or `*` for bullet lists, `1.` `2.` `3.` for numbered lists.

### Links

Write `[Link text](https://example.com)` to create a link.

### Code

Use single backticks for inline code, triple backticks for code blocks.

---

## Spoiler Text

Hide text that reveals on hover ‚Äî perfect for hiding answers!

**Syntax:** Write two vertical bars, then your text, then two more vertical bars.

Example: `| |hidden text| |` (without spaces between bars)

**Tip:** Use **Ctrl+H** to quickly wrap selected text in spoiler tags.

## Flashcards

Create flashcards directly in your notes for spaced repetition learning!

### Syntax

Write `??` then your question, then `::` then your answer, then `??`

Format: `??Question::Answer??`

### Examples

- `??What is the capital of France?::Paris??`
- `??What does HTML stand for?::HyperText Markup Language??`
- `??Formula for area of a circle::A = œÄr¬≤??`

### Multi-line Cards

You can write questions and answers on multiple lines between the `??` markers.

### Study Mode

1. Click the **üìö Study** button in the toolbar
2. View the question
3. Click **Show Answer** to reveal
4. Rate your knowledge (1-5)
5. Cards are scheduled based on your ratings using the SM-2 algorithm

### Tips

- Add flashcards anywhere in your notes
- Use spoilers in answers for partial hints
- Study regularly for best retention
- Check "Due for review today" widget in the sidebar
